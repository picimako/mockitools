Can perform conversion of one or more Mockito.verify() call chains to the following approaches:
  • to InOrder.verify(),
  • to BDDMockito.then() using a newly created an InOrder local variable,
  • to BDDMockito.then() without an InOrder, omitting the source InOrder variable.

The intention is available on Mockito.verify() when it is followed by a method call on the mock object.
