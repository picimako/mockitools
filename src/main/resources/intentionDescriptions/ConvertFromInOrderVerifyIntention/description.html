Can perform conversion of one or more InOrder.verify() call chains to the following approaches:
  • to Mockito.verify(),
  • to BDDMockito.then() using a newly created an InOrder local variable,
  • to BDDMockito.then() without an InOrder, omitting the source InOrder variable.
  • to MockedStatic.verify()

The intention is available on InOrder.verify() when it is followed by a method call (only in case of non-MockedStatic) on the mock object.

In call chains where there is no separate InOrder local variable is created, but Mockito.inOrder(mockObject) is called inline, is not supported at the moment.

In any other case, the option list displays a No action available message.
