Can perform conversion of one or more BDDMockito.then().should() call chains to the following approaches:
  • to Mockito.verify() (omits the InOrder arguments),
  • to BDDMockito.then() using a newly created an InOrder local variable, if there is none specified in the should() call(s) yet,
  • to InOrder.verify() by either creating a new InOrder variable, or reusing the one specified in the should() call

The intention is available on BDDMockito.then() when it is followed by should() and a method call on the mock object.
