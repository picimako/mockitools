This intention action expands certain mock/spy creation calls to use concrete MockSettings configuration, and aims to simplify the process of converting mock creation logic when further mock settings need to be added.

The following conversions/expansions are supported: 


//From:
Mockito.spy(<spiedInstance>)
//to:
Mockito.mock(<raw type of spiedInstance>, withSettings().spiedInstance(<spiedInstance>))

//From:
Mockito.mock(<type>.class, <name>)
//to:
Mockito.mock(<type>.class, withSettings().name(<name>))

//From:
Mockito.mock(<type>.class, <answer>)
//to:
Mockito.mock(<type>.class, withSettings().defaultAnswer(<answer>))
