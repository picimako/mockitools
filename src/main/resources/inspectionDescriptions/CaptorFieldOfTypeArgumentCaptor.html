Fields that are annotated as @Captor must have ArgumentCaptor as their type, otherwise Mockito will fail with a message saying the type must be ArgumentCaptor.

This inspection reports fields that are annotated as @Captor but their types are not ArgumentCaptor.

It also provides a quick fix to convert the field type to ArgumentCaptor with the appropriate generic type. Some examples (before-after): 

    
//-- arrays --

@Captor
String[] captor;
@Captor
ArgumentCaptor<String[]> captor;

//-- primitive types --

@Captor
char captor;
@Captor
ArgumentCaptor<Character> captor;

//-- types with generic types --

@Captor
List<List<String>> captor;
@Captor
ArgumentCaptor<List<List<String>>> captor;
