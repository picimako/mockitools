Reports usages of JUnit runners used from the org.mockito.runners package, which is removed in Mockito v4.

Quick fixes can be applied to replace these classes

  • ConsoleSpammingMockitoJUnitRunner
  • VerboseMockitoJUnitRunner
  • MockitoJUnitRunner
to org.mockito.junit.MockitoJUnitRunner.

The inspection is applied only when the Mockito version included on the module classpath of the currently inspected file is 2.x or 3.x. 


//from:
import org.junit.runner.RunWith;
import org.mockito.runners.ConsoleSpammingMockitoJUnitRunner;

@RunWith(ConsoleSpammingMockitoJUnitRunner.class)
public class JUnitRunnerTest {
}

//to:
import org.junit.runner.RunWith;
import org.mockito.runners.ConsoleSpammingMockitoJUnitRunner; //this import is left in for now
import org.mockito.junit.MockitoJUnitRunner;

@RunWith(MockitoJUnitRunner.class)
public class JUnitRunnerTest {
}
