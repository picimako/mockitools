Reports exception references in *throw() stubbing methods based on Mockito's rule on checked exceptions. If [the specified exception types] contain a checked exception then it has to match one of the checked exceptions of method signature.

The following constructs are supported:

  • Mockito.when().thenThrow(...)
  • BDDMockito.given().willThrow(...)
  • Mockito.doThrow(...).when()
  • BDDMockito.willThrow(...).given()

In case of each way of stubbing, further chained *throw() calls are supported too. In case of an empty list, no problem is reported.
