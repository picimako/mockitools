Reports multiple consecutive calls to *Throw() methods, so that they may be merged into a single call.

Both org.mockito.Mockito and org.mockito.BDDMockito based stubbing chains are supported, including calls to doThrow(), thenThrow() and willThrow().

If there are multiple sections of consecutive calls within the same call chain, they are reported separately for better notification, and can be merged separately.

Every parameter combination of *Throw() calls can be merged but

  • when there are both AnException.class and new AnException() type arguments (as default constructor calls), users can choose which type to convert the expressions to,
  • when there is a new AnException("reason") type call to a parameterized constructor, the arguments are automatically converted to Throwables to keep the constructor arguments.
