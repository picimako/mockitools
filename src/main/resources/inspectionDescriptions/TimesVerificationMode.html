Reports Mockito.times() calls that can be optimized or deleted based on their argument values.
  • times(0) calls can be replaced with Mockito.never() for better readability,
  • times(1) calls can be removed since 1 is the default value when using times() verification. See Mockito exact verification documentation.

Quick fixes are also provided for the replacement and removal of these calls. 


//before - times(1) can be omitted
Mockito.verify(mock, times(1))...
//after
Mockito.verify(mock)...

//no quick fix, left untouched
Mockito.verify(mock, times(1).description("message"))...

//before - times(0) can be replaced with never()
Mockito.verify(mock, times(0))...
//after
Mockito.verify(mock, never())...

//before - times(0) can be replaced with never()
Mockito.verify(mock, times(0).description("message"))...
//after
Mockito.verify(mock, never().description("message"))...
