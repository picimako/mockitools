This inspection reports spy creation on mock objects, for example 

Mockito.spy(Mockito.mock(...))
or 

Mockito.spy(<object annotated with @Mock>)

The corresponding feature was introduced in Mockito 5.4.0, but this inspection does not do a library version check, and validates test code regardless of the Mockito version.

NOTE: variables with Mockito.mock() initializers are not recognized as mocks yet.
