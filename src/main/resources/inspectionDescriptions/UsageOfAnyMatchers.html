Reports usages of certain anyX() matchers which are deprecated since Mockito v2, and are removed in v4.

Accordingly, it also provides quick fixes

  • to replace anyObject() and anyVararg() with any(),
  • to replace anyCollectionOf(), anyIterableOf(), anyListOf(), anyMapOf() and anySetOf() with their anyX() variants.

The inspection is applied only when the Mockito version included on the module classpath of the currently inspected file is 2.x or 3.x.

anyObject/anyVararg -> any

    
//from:
Mockito.when(mock.method(ArgumentMatchers.anyVararg())).thenReturn(10);
//to:
Mockito.when(mock.method(ArgumentMatchers.any())).thenReturn(10);

anyXOf -> anyX

    
//from:
Mockito.when(mock.method(ArgumentMatchers.anyCollectionOf(String.class))).thenReturn(10);
//to:
Mockito.when(mock.method(ArgumentMatchers.anyCollection())).thenReturn(10);

Furthermore, if the matcher is referenced via org.mockito.Matchers, it is also replaced with the org.mockito.ArgumentMatchers qualifier.
